Computer graphics using C++

Daya
March 5, 2023

Welcome to my blog series on computer graphics with C++ and OpenGL! Whether you're a beginner or an experienced programmer, this series is designed to help you learn how to create stunning 3D graphics and animations using industry-standard tools and techniques.

Throughout this series, we'll cover the fundamentals of computer graphics, including how to render 3D objects, apply textures and lighting, and implement advanced techniques like shadow mapping. We'll also explore the basics of OpenGL, one of the most widely-used graphics APIs, and how to use it to create powerful and dynamic graphics applications.

By the end of this series, you'll have a solid understanding of the key concepts and tools used in modern computer graphics programming, as well as the ability to apply this knowledge to create your own amazing graphics projects. So whether you're looking to build your own games, create stunning visualizations, or simply learn more about the exciting world of computer graphics, this series is for you!

Prerequisites

  1. A reasonably modern computer that supports OpenGL, C++ compiler and ide
  2. Libraries
    • GLAD ( OpenGL loader library )
    • GLM ( Math library )
    • GLFW ( Window creation library )

Setting up the project:

You can either create a basic C++ project by yourself, and include all the required libraries, or use the provided starter project

Using the provided starter project:

StartupProject.zip

Visual Studio 2017+

  1. Extract the StarterProject zip file
  2. open it in visual studio

CMake Command Line:

  1. Extract the StarterProject zip file
  2. Open the extracted folder
  3. Create a build folder
  4. Open terminal here
  5. To Build: type "cmake --build .." from the build folder

Now you should be ready to start!
